 ## 함수 (Function)

 <br>

  ### 함수 표현식 : 익명 함수를 선언하고, 변수에 대입 
  ### 함수 선언식 : 함수명과 함께 함수를 선언 
  ### 고차 함수 : 함수가 함수를 리턴하거나 인자로 받는 패턴 
  ```js
  /// 고차함수 예시

  function a(){
    console.log('a');
    function b(){
      console.log('b');
    }
    return b;
  }
  ```
  ### 콜백함수 : 고차함수 패턴 중, 매개변수로 함수를 넘겨받아호출하는 패턴
